# Assets

This directory contains diagrams, images, and other visual assets for the `mobot_core` documentation.

---

## Diagram Tools

Recommended tools for creating diagrams:

- **draw.io** (free, web-based): [https://app.diagrams.net](https://app.diagrams.net)
- **Excalidraw** (free, hand-drawn style): [https://excalidraw.com](https://excalidraw.com)
- **Mermaid** (Markdown-embeddable): [https://mermaid.js.org](https://mermaid.js.org)
- **TikZ** (LaTeX diagrams): Embedded in `latex/main.tex`

---

## File Naming Convention

Please use lowercase and underscores for all asset filenames, e.g.:
- `system_overview.png` ✅
- `SystemOverview.PNG` ❌
